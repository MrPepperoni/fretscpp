Where to get songs


Original Frets-on-Fire

Sectoid Songs (Ubuntu archive)
Muldjord Songs (Ubuntu archive)

The data/songs directory from the archives must end up 
in the data/songs directory of fretscpp under Windows
and in 
/usr/share/games/fretscpp/data/songs under Linux 
(or in "HOME/.config/fretscpp/data/songs" if you don't have write access)

Under Windows you can open .tar.gz files using 7zip or WinRAR.
